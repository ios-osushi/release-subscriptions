  - `releases`（GitHub の「リリース」）
  - `tags`（GitHub の「タグ」）
- `name`: リポジトリで公開されているソフトウェアの名前
- `owner`: リポジトリのオーナー
- `repo`: リポジトリの名称

列挙の順番（ソート順）のルールは以下のとおりです。

1. `kind` が `primary` → `secondary` の順
1. `owner` の大文字小文字区別なし昇順
1. `repo` の大文字小文字区別なし昇順
